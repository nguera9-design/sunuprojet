## 🚀 Installation

### Prérequis
- XAMPP (Apache, MySQL, PHP 8+)

### Étapes
1. Copier le dossier dans `C:\xampp\htdocs\`
2. Lancer XAMPP (Apache + MySQL)
3. Créer la base `sunuprojet` dans phpMyAdmin
4. Importer `sunuprojet.sql`
5. Ouvrir : http://localhost/sunuprojet/login.php
